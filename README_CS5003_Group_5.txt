1. Open MongoDB Compass, connect mongodb://localhost:27017 and click "Connect".
2. Go to the server folder and copy the directory address.
3. Open the Terminal and enter "cd <address>".
4. Enter "node app.js" or "nodemon app.js" to connect the server.
5. Copy the URL displayed in the terminal.
6. Open a web browser and paste the URL to open the web application "Social Runner".
